保存博客文件

新建一个github仓库保存全部博客文件，记得把主题的.git文件删掉，如此才能把它push上去,还有要把.gitignore清空保证全部文件都能push

利用github action 实现自动部署

密钥部署

创建一个新ssh密钥用于部署

登录GitHub网站，找到博客源码仓库；添加新的Secrets，在：Settings -> Secrets -> Action：

  1. Name填写：HEXO_DEPLOY_SECRET，下边会用到；
  2. Value填写：hexo-deploy-key(私钥)文件中的全部内容；

找到静态页面仓库；添加新的Deploy key，在：Settings -> Deploy keys：

  1. Title随便填写；
  2. Value填写：hexo-deploy-key.pub(公钥)文件中的全部内容；
  3. 勾选“Allow Write Access”；

也可以直接用连接github用的ssh密钥，这样只需要改博客文件仓库的Secret，不用改page仓库的deploy key了。
